Locked down

Secure-by-design, in plain terms.

🔐 Nobody walks in
  • Real @colt.net email + shared password + a one-time code emailed to that inbox. Guessing the first two isn't enough.
🙈 Secrets never in git
  • Keys live only on the server or as encrypted GitHub secrets; gitleaks blocks accidental commits.
🧪 Scanned before ship
  • Trivy (deps+image), CodeQL SAST, ruff, pytest — every change checked before it reaches the server.
🚧 Never breaks the neighbours
  • Isolated colt-stack; the existing VPN & services and the firewall are untouched.

It heals & ships itself

A server nobody patches gets hacked; a change nobody ships is useless. Both are automatic.

💾Back up first

Every 3 days it snapshots to DO Spaces before touching anything. Backup fails → no upgrade.

⬆️Patch + AI digest

Updates OS/Docker; DeepSeek writes a plain-English summary flagging kernel/CVE risks. Reboots at 4am.

🚀One-command ship

python ship.py or git push → build, Trivy-scan, deploy, verify — over a private Tailscale tunnel.
