What happens on one /assess

Authenticate

AE sends /auth name.familyname@colt.net <pass>. Email + shared password checked (constant-time).

02

Prove the mailbox

A 6-digit code is emailed via the Gmail API over HTTPS; AE replies /verify 123456. Guessing email+password isn't enough.

03

Recon

colttechbot runs the Top-10 Shodan filters + RIPE/RDAP, resolving the estate across ASNs, orgs, certs & hostnames.

04

Enrich
