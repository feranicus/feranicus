// 03 вҖ” Why an open n8n is a big deal

To do that it stores the credentials for everything it touches, and its workflows can run logic вҖ” a built-in code/execute node, HTTP calls and public webhooks. So an exposed admin surface isn't "just a login page" вҖ” it's code execution plus a master key ring, handed to whoever walks up.
